# 镜头靠近转场效果说明

## 功能介绍
点击建筑物时，镜头会先向建筑物靠近，同时屏幕渐渐变黑，然后切换到目标场景。

## 效果流程

1. 点击建筑物
2. 镜头开始向建筑物移动（1.5秒）
3. 移动到30%时，屏幕开始渐渐变黑
4. 镜头到达建筑物附近，屏幕完全变黑
5. 切换到新场景
6. 屏幕从黑色渐渐变亮

## 实现原理

### 镜头移动
- 计算从当前位置到建筑物的方向
- 镜头移动到建筑物前方3米处（可配置）
- 镜头自动旋转朝向建筑物
- 使用平滑曲线（SmoothStep）让移动更自然

### 同步淡出
- 镜头移动到30%时开始淡出
- 淡出和移动同步进行
- 到达目标时屏幕完全变黑

### 自动禁用控制
- 转场期间自动禁用镜头控制器
- 防止用户操作干扰转场动画
- 场景切换后恢复控制

## 配置参数

在SceneTransitionManager的Inspector中：

### 镜头移动设置
- Camera Move Time: 镜头移动持续时间（默认1.5秒）
- Camera Distance Offset: 镜头距离建筑物的偏移（默认3米）

### 淡入淡出设置
- Fade Duration: 淡入淡出持续时间（默认1秒）
- Fade Color: 淡出颜色（默认黑色）

## 调整建议

### 快速转场
- Camera Move Time: 1秒
- Camera Distance Offset: 2米
- 适合小型建筑或快节奏游戏

### 慢速转场（电影感）
- Camera Move Time: 2-3秒
- Camera Distance Offset: 5米
- 适合大型建筑或强调氛围

### 近距离转场
- Camera Move Time: 1.5秒
- Camera Distance Offset: 1-2米
- 镜头会非常靠近建筑物

### 远距离转场
- Camera Move Time: 2秒
- Camera Distance Offset: 5-8米
- 保持一定距离，更全面地展示建筑

## 使用说明

### 自动应用
- ClickableBuildingGroup已自动更新
- 无需修改任何设置
- 点击建筑物即可看到新效果

### 手动调用
如果需要通过代码触发：

```csharp
// 获取建筑物位置
Vector3 buildingPosition = building.transform.position;

// 使用场景名称切换
SceneTransitionManager.Instance.TransitionToSceneWithCamera("SceneName", buildingPosition);

// 使用场景索引切换
SceneTransitionManager.Instance.TransitionToSceneWithCamera(1, buildingPosition);
```

### 不使用镜头移动
如果某些情况下不想要镜头移动效果：

```csharp
// 使用普通的场景切换（无镜头移动）
SceneTransitionManager.Instance.TransitionToScene("SceneName");
```

## 技术细节

### 目标位置计算
- 优先使用建筑物Renderer的bounds中心
- 如果没有Renderer，使用transform.position
- 确保镜头对准建筑物的视觉中心

### 镜头路径
- 起点：当前镜头位置
- 终点：建筑物前方（距离由Camera Distance Offset决定）
- 路径：直线插值，使用SmoothStep平滑

### 旋转处理
- 起始旋转：当前镜头旋转
- 目标旋转：LookRotation朝向建筑物
- 插值：Quaternion.Slerp球面插值

### 淡出时机
- 前30%：只移动镜头，不淡出
- 后70%：同时移动镜头和淡出
- 这样可以先看到靠近的过程，再变黑

## 常见问题

### 问题1：镜头移动太快/太慢
- 调整Camera Move Time参数
- 建议范围：1-3秒

### 问题2：镜头离建筑物太近/太远
- 调整Camera Distance Offset参数
- 建议范围：2-8米

### 问题3：淡出开始太早/太晚
- 这是硬编码在30%开始
- 如需修改，编辑MoveCameraToTarget方法中的0.3f值

### 问题4：镜头移动不平滑
- 检查帧率是否稳定
- 确保没有其他脚本干扰镜头

### 问题5：转场时用户还能控制镜头
- 脚本会自动禁用SceneViewCameraController
- 如果使用其他镜头控制器，需要手动禁用

## 效果对比

### 旧版（直接淡出）
1. 点击建筑
2. 屏幕变黑
3. 切换场景
4. 屏幕变亮

### 新版（镜头靠近）
1. 点击建筑
2. 镜头向建筑移动
3. 移动过程中屏幕渐渐变黑
4. 切换场景
5. 屏幕变亮

新版提供了更好的空间感和沉浸感！

## 性能说明

- 镜头移动使用Lerp和Slerp，性能开销极小
- 不会影响场景加载速度
- 适合在任何平台使用

## 扩展建议

### 添加音效
在点击建筑时播放音效：
```csharp
AudioSource.PlayClipAtPoint(clickSound, transform.position);
```

### 添加粒子效果
在镜头移动时显示粒子：
```csharp
ParticleSystem.Play();
```

### 自定义移动曲线
修改MoveCameraToTarget中的插值方式，使用AnimationCurve自定义曲线。
