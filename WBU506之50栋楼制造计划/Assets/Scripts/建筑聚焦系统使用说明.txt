# 建筑聚焦系统使用说明

## 功能介绍
点击建筑物后，隐藏其他建筑物，镜头聚焦到该建筑物正面。按A键恢复场景。

## 效果流程

### 点击建筑物时
1. 其他建筑物慢慢淡出消失
2. 镜头平滑移动到建筑物正面
3. 镜头自动旋转朝向建筑物
4. 只剩下被点击的建筑物可见

### 按A键恢复时
1. 其他建筑物慢慢淡入显现
2. 镜头平滑回到原始位置
3. 镜头恢复原始角度
4. 重新启用镜头控制

## 使用步骤

### 1. 添加BuildingFocusManager
- 在场景中创建空物体（GameObject → Create Empty）
- 命名为 "BuildingFocusManager"
- 添加 BuildingFocusManager 脚本

### 2. 配置参数

**聚焦设置**
- Focus Duration: 聚焦动画时长（默认1.5秒）
- Focus Distance: 镜头距离建筑物的距离（默认10米）
- Height Offset: 镜头高度偏移（默认2米）

**淡入淡出设置**
- Fade Duration: 建筑物淡入淡出时长（默认1秒）

**自动检测**
- Auto Find Buildings: 自动查找所有建筑物（推荐勾选）
- Buildings: 手动指定建筑物列表（可选）

### 3. 设置建筑物

选择每个建筑物的ClickableBuildingGroup组件：
- Click Behavior: 选择 "Focus Building"
- 其他参数保持不变

### 4. 测试
- 运行游戏
- 点击任意建筑物 → 聚焦效果
- 按A键 → 恢复场景

## 两种模式对比

### Focus Building（聚焦建筑）- 新功能
- 点击后隐藏其他建筑
- 镜头聚焦到建筑物
- 按A键恢复
- 不切换场景

### Scene Transition（场景切换）- 原功能
- 点击后切换到其他场景
- 镜头靠近建筑物
- 屏幕淡出
- 加载新场景

## 配置建议

### 小型建筑
- Focus Distance: 5-8米
- Height Offset: 1-2米
- Focus Duration: 1秒

### 中型建筑
- Focus Distance: 10-15米
- Height Offset: 2-3米
- Focus Duration: 1.5秒

### 大型建筑
- Focus Distance: 15-25米
- Height Offset: 3-5米
- Focus Duration: 2秒

## 工作原理

### 自动检测建筑物
- 查找所有带ClickableBuildingGroup的物体
- 自动收集它们的Renderer组件
- 建立管理列表

### 聚焦过程
1. 保存当前镜头位置和角度
2. 禁用镜头控制器
3. 计算目标镜头位置（建筑物正面）
4. 同时执行：
   - 镜头平滑移动到目标位置
   - 其他建筑物淡出（透明度0）

### 恢复过程
1. 同时执行：
   - 镜头平滑回到原始位置
   - 其他建筑物淡入（透明度1）
2. 重新启用镜头控制器
3. 清除聚焦状态

### 透明度处理
- 使用材质的_Color属性设置透明度
- 透明度为0时禁用Renderer（性能优化）
- 自动切换到透明渲染模式

## 常见问题

### 问题1：点击建筑物没反应
- 检查是否添加了BuildingFocusManager
- 确保Click Behavior设置为Focus Building
- 查看Console是否有错误信息

### 问题2：其他建筑物没有消失
- 确保Auto Find Buildings被勾选
- 检查建筑物是否有ClickableBuildingGroup组件
- 查看Console的建筑物数量日志

### 问题3：按A键没反应
- 确保已经处于聚焦状态
- 检查是否使用了新的Input System
- 查看Console是否有错误

### 问题4：镜头位置不对
- 调整Focus Distance参数
- 调整Height Offset参数
- 检查建筑物的位置和大小

### 问题5：建筑物变成透明的
- 这是正常的淡出效果
- 如果恢复后还是透明，检查材质设置
- 尝试重新运行游戏

## 高级用法

### 通过代码控制

```csharp
// 聚焦到指定建筑物
GameObject building = GameObject.Find("Building1");
BuildingFocusManager.Instance.FocusOnBuilding(building);

// 取消聚焦
BuildingFocusManager.Instance.UnfocusBuilding();

// 检查是否处于聚焦状态
bool isFocused = BuildingFocusManager.Instance.IsFocused();

// 获取当前聚焦的建筑物
GameObject focused = BuildingFocusManager.Instance.GetFocusedBuilding();
```

### 自定义按键
修改BuildingFocusManager的Update方法，更改按键：

```csharp
// 改为按ESC键恢复
if (keyboard.escapeKey.wasPressedThisFrame)
{
    UnfocusBuilding();
}
```

### 手动指定建筑物列表
如果不想自动检测：
1. 取消勾选Auto Find Buildings
2. 手动拖拽建筑物到Buildings列表

### 添加UI提示
在聚焦时显示提示文字：

```csharp
// 在FocusCoroutine中添加
UIManager.ShowText("按A键返回");

// 在UnfocusCoroutine中添加
UIManager.HideText();
```

## 性能优化

### 已实现的优化
- 透明度为0时禁用Renderer
- 使用材质实例避免影响其他物体
- 平滑插值使用SmoothStep

### 建议
- 建筑物数量不要超过50个
- 每个建筑物的子物体不要超过100个
- 使用LOD系统优化远处建筑物

## 注意事项

1. 确保所有建筑物都有ClickableBuildingGroup组件
2. 确保建筑物的材质支持透明度
3. 聚焦状态下无法控制镜头（按A键恢复后才能控制）
4. 同一时间只能聚焦一个建筑物
5. 建筑物的Collider要覆盖整个建筑

## 故障排除

### 建筑物没有被检测到
- 检查是否有ClickableBuildingGroup组件
- 确保Auto Find Buildings被勾选
- 查看Console的日志信息

### 透明度效果不正常
- 检查材质是否支持透明度
- 确保材质有_Color属性
- 尝试使用Standard Shader

### 镜头移动不平滑
- 增加Focus Duration的值
- 检查帧率是否稳定
- 确保没有其他脚本干扰镜头

## 效果展示

### 聚焦前
- 场景中所有建筑物可见
- 镜头可以自由移动
- 可以点击任意建筑物

### 聚焦后
- 只有被点击的建筑物可见
- 镜头固定在建筑物正面
- 无法控制镜头（需要按A键恢复）

### 恢复后
- 所有建筑物重新显现
- 镜头回到原始位置
- 可以继续自由操作
