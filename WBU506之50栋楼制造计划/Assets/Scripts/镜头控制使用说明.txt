# 镜头控制器使用说明

## 功能介绍
CameraController 脚本实现了以下功能：
1. 鼠标右键或中键拖动 - 旋转镜头视角
2. 鼠标滚轮 - 缩放镜头距离
3. 自动围绕目标点旋转

## 使用步骤

### 1. 添加脚本到摄像机
- 在 Hierarchy 中选择 Main Camera
- 将 CameraController.cs 脚本拖到 Main Camera 上
- 或者点击 Add Component，搜索 "CameraController"

### 2. 配置参数（在 Inspector 面板）

【旋转设置】
- Rotation Speed: 旋转速度（默认 5）
- Min Vertical Angle: 最小垂直角度（默认 -80）
- Max Vertical Angle: 最大垂直角度（默认 80）

【缩放设置】
- Zoom Speed: 缩放速度（默认 10）
- Min Zoom: 最小缩放距离（默认 2）
- Max Zoom: 最大缩放距离（默认 20）

【目标设置】
- Target: 镜头围绕的目标物体（可选，留空则自动创建中心点）
- Initial Distance: 初始距离（默认 10）

### 3. 操作方式
- 按住鼠标右键或中键拖动 → 旋转视角
- 滚动鼠标滚轮 → 拉近/拉远镜头

### 4. 可选：设置目标物体
如果想让镜头围绕特定物体旋转：
- 在场景中创建一个空物体（GameObject → Create Empty）
- 将该物体拖到 CameraController 的 Target 槽位
- 调整该物体的位置作为旋转中心

## 代码调用示例

```csharp
// 获取镜头控制器
CameraController cameraController = Camera.main.GetComponent<CameraController>();

// 设置新的目标位置
cameraController.SetTarget(new Vector3(0, 0, 0));

// 重置镜头
cameraController.ResetCamera();
```

## 注意事项
1. 确保场景中只有一个 Main Camera
2. 如果镜头行为异常，检查 Target 是否正确设置
3. 可以在运行时通过代码动态调整参数
