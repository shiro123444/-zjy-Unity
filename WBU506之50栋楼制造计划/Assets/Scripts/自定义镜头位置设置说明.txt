# 自定义镜头位置设置说明

## 功能介绍
为每个建筑物设置自定义的镜头目标位置和角度，精确控制聚焦效果。

## 设置步骤

### 方法1：使用空物体作为目标（推荐）

1. **创建镜头目标物体**
   - 在建筑物下创建空物体：右键建筑物 → Create Empty
   - 命名为 "CameraTarget"

2. **调整目标位置**
   - 在Scene视图中移动空物体到理想的镜头位置
   - 旋转空物体，让Z轴（蓝色箭头）指向建筑物
   - 这就是镜头将要移动到的位置和角度

3. **关联到建筑物**
   - 选择建筑物
   - 找到ClickableBuildingGroup组件
   - 将CameraTarget拖到Custom Camera Target槽位

4. **测试**
   - 运行游戏
   - 点击建筑物
   - 镜头会移动到你设置的位置

### 方法2：使用现有摄像机位置

1. **调整Scene视图镜头**
   - 在Scene视图中调整到理想角度
   - 选择Main Camera
   - GameObject → Align With View（Ctrl+Shift+F）

2. **记录位置**
   - 记下Camera的Position和Rotation值

3. **创建目标物体**
   - 创建空物体
   - 手动输入记录的Position和Rotation值
   - 关联到建筑物

### 方法3：使用辅助脚本（高级）

创建一个编辑器脚本来快速设置：

```csharp
[MenuItem("Tools/Set Camera Target From View")]
static void SetCameraTargetFromView()
{
    GameObject target = new GameObject("CameraTarget");
    target.transform.position = SceneView.lastActiveSceneView.camera.transform.position;
    target.transform.rotation = SceneView.lastActiveSceneView.camera.transform.rotation;
    Selection.activeGameObject = target;
}
```

## 调整技巧

### 找到最佳角度

1. **使用Scene视图预览**
   - 在Scene视图中调整到满意的角度
   - 选择CameraTarget空物体
   - GameObject → Align With View

2. **微调位置**
   - Position X: 左右移动
   - Position Y: 上下移动
   - Position Z: 前后移动

3. **微调角度**
   - Rotation X: 俯仰角度
   - Rotation Y: 水平旋转
   - Rotation Z: 倾斜角度（通常保持0）

### 常用视角设置

**正面视角**
- Position: (0, 2, -10) 相对于建筑物
- Rotation: (0, 0, 0)

**侧面视角**
- Position: (10, 2, 0) 相对于建筑物
- Rotation: (0, -90, 0)

**俯视视角**
- Position: (0, 15, -5) 相对于建筑物
- Rotation: (45, 0, 0)

**仰视视角**
- Position: (0, -2, -10) 相对于建筑物
- Rotation: (-15, 0, 0)

## 可视化辅助

### 在Scene视图中显示目标

CameraTarget空物体在Scene视图中会显示为：
- 一个小图标
- 蓝色箭头（Z轴）指向镜头朝向

### 添加Gizmo显示

可以给CameraTarget添加一个脚本来显示视锥：

```csharp
void OnDrawGizmos()
{
    Gizmos.color = Color.yellow;
    Gizmos.DrawWireSphere(transform.position, 0.5f);
    Gizmos.DrawRay(transform.position, transform.forward * 5f);
}
```

## 多个建筑物设置

### 批量设置相同角度
1. 创建一个CameraTarget模板
2. 复制该模板到每个建筑物下
3. 调整每个副本的位置（保持角度相同）

### 不同建筑物不同角度
- 每个建筑物单独设置CameraTarget
- 根据建筑物朝向调整角度

## 自动计算 vs 自定义

### 留空Custom Camera Target（自动计算）
- 系统自动计算位置
- 位置：建筑物正面
- 距离：由Focus Distance参数控制
- 高度：由Height Offset参数控制

### 设置Custom Camera Target（自定义）
- 使用指定的位置和角度
- 完全自定义控制
- 忽略Focus Distance和Height Offset参数

## 常见问题

### 问题1：镜头位置不对
- 检查CameraTarget的Position值
- 确保CameraTarget在正确的位置
- 在Scene视图中查看CameraTarget的位置

### 问题2：镜头角度不对
- 检查CameraTarget的Rotation值
- 确保Z轴（蓝色箭头）指向建筑物
- 使用Align With View对齐

### 问题3：看不到建筑物
- CameraTarget可能太近或太远
- 调整Position Z值
- 确保镜头朝向建筑物

### 问题4：镜头倾斜
- 检查Rotation Z值
- 通常应该保持为0
- 重置Rotation为合适的值

## 调试技巧

### 显示目标位置
在运行时在Console中查看：
```
使用自定义镜头位置: (x, y, z)
```

### 测试不同位置
1. 运行游戏
2. 暂停
3. 在Scene视图中移动CameraTarget
4. 继续运行
5. 观察效果

### 快速迭代
1. 设置CameraTarget
2. 运行游戏测试
3. 停止游戏
4. 调整CameraTarget
5. 重复

## 最佳实践

1. **命名规范**
   - 使用清晰的命名：Building1_CameraTarget
   - 保持层级结构清晰

2. **位置管理**
   - 将CameraTarget作为建筑物的子物体
   - 便于整体移动建筑物

3. **备份设置**
   - 记录满意的Position和Rotation值
   - 方便后续调整

4. **统一风格**
   - 保持相似建筑物的镜头角度一致
   - 提供统一的视觉体验

## 示例设置

- Position: (0, 3, -15)
- Rotation: (10, 0, 0)
- 效果：略微俯视，看到建筑物全貌

### 宿舍楼
- Position: (8, 2, -8)
- Rotation: (5, -45, 0)
- 效果：侧前方视角，更有立体感

### 图书馆
- Position: (0, 5, -20)
- Rotation: (15, 0, 0)
- 效果：远距离俯视，展示建筑物规模
